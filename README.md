# Survey Research Toolkit (UK)

This is a production-grade, multi-tenant SaaS foundation for UK surveyors and valuers.

## Stack
- Next.js 15 (App Router)
- Prisma 7 (PostgreSQL)
- Auth.js (v5)
- Tailwind CSS & shadcn/ui

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
